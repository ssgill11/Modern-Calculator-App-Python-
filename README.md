# 🧮 Modern Calculator App (Python)
## 📌 Overview

A clean and responsive desktop calculator built using **Python** and **Tkinter**, featuring a modern UI with perfectly aligned buttons and smooth interaction.

---

## 🚀 Features

* ➕ Basic arithmetic operations (+, −, ×, ÷)
* 🎯 Clean and aligned button layout using grid system
* 🌙 Modern dark-themed user interface
* ⚡ Fast and lightweight (no external dependencies)
* 🧹 Clear button to reset calculations
* ❗ Error handling for invalid inputs

---

## 🛠️ Technologies Used

* Python
* Tkinter

## ▶️ How to Run

### 1. Install Python

Download and install Python from the official website.

### 2. Run the Application

Open terminal or VS Code and run:

```
python calculator.py
```

or

```
python3 calculator.py
```

---

## 💻 Requirements

* Python 3.x
* Tkinter (comes pre-installed with Python)

To verify Tkinter:

```
python -m tkinter
```

## ⚠️ Note

This app uses Python's built-in `eval()` function for calculations.
It is suitable for learning purposes but not recommended for production-level applications due to security concerns.
